# Día 1 - Estructura y primeras funcionalidades

## 🧠 Qué he trabajado
- Creación de una web básica con HTML, CSS y JavaScript
- Uso de un servidor local con Python
- Lectura de datos desde un archivo JSON

## 🔧 Funcionalidades implementadas
- Input para introducir código postal
- Botón de búsqueda
- Lectura de estaciones desde `stations.json`
- Filtrado de gasolineras por código postal
- Mostrar resultados dinámicamente en pantalla

## 🧩 Problemas encontrados
- Dudas sobre estructura del proyecto
- Comprensión de objetos JSON
- Cómo recorrer arrays en JavaScript

## ✅ Cómo los he solucionado
- Uso de `fetch()` para leer JSON
- Uso de `.filter()` para buscar coincidencias
- Uso de `forEach()` para pintar resultados

## 💡 Conceptos aprendidos
- Diferencia entre HTML, CSS y JS
- Uso de arrays y objetos en JavaScript
- Métodos básicos: `filter`, `forEach`
- Relación entre frontend y datos
